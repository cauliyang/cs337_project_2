## How to use

```sh
# clone the repo
git clone https://github.com/cauliyang/cs337_project_1.git

# install uv
pip install uv

# install dependencies
uv sync

```

## Development

```sh
# run the tests
uv run pytest -vls
# or
make test

# add Dependency
uv add requests

# run the linter before pushing
make lint
```
